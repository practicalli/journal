## Cycling

Continuing to experiment with GPStitch to create overlays from GPX / Fit files.

Wahoo doesnt make it easy to find the FIT files from previous rides via their website.

The FIT files can be downloaded from the Wahooligan site, but you have to know the actual URL as there isn't a way to search for those files.

Luckily I kept all the emails that Wahoo sent me, each email contains a link to download the FIT file.


### GPStitch

GPStitch video overlay was generated from the Dragon Ride FIT file from Wahoo.

After a little experimentation, I found that **Compositing** > **Blend** > **Lighten** setting gave the better results when on top of a camera video.

I edited the default GPStitch layout for the overlay:

- repositioned data metrics so speed, heart rate and cadence are all together
- elevation, gradient and altitude under the maps
- maps moved down and set to an opacity of 1 (default 0.7 has a tendency to disappear on lighter backgrounds, e.g. sky)
