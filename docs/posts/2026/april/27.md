## Alternative Clojure.org

I started working on an alternative design for the Clojure.org website.  There is a lot of highly valuable information on this website, although it is not always the easiest to navigate.  Information discovery is some times a challenge, limiting the Clojure adoption potential that this website should provide.

The purpose of the [Alt Clojure.org](https://practical.li/alternative-clojure-org) website is to experiment with content, design and UX ideas for enhancing and extending the Clojure.org website.

This project is not currently sponsored by Nubank or with any collaboration from the Clojure core team (although both would be welcome).

A secondary purpose is to evaluate the capabilities of [Zensical](https://zensical.org/). This a fast static site generator with a rich author experience to create highly engaging websites.

> NOTE: Zensical is under very active development and already provides a good experience.  Adoption is growing and feature parity with [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/) is expected later in 2026.
